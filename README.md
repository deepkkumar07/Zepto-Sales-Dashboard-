# # Zepto Sales Analytics Dashboard | Power BI

## Project Overview

This project is an interactive Power BI dashboard developed to analyze Zepto's sales performance, customer preferences, outlet efficiency, and product category trends. The dashboard transforms raw business data into actionable insights through advanced data modeling, DAX calculations, and interactive visualizations.

## Key Business Insights

* Monitored Total Sales, Total Items Sold, Average Sales, and Average Customer Ratings.
* Analyzed sales performance across different outlet locations and outlet sizes.
* Evaluated product category contribution to overall revenue.
* Compared outlet types based on sales, ratings, and item distribution.
* Identified historical sales trends through outlet establishment analysis.
* Examined customer preferences based on fat content categories and product types.

## Power BI Features Implemented

* Data Cleaning and Transformation using Power Query
* Data Modeling with Relationships
* DAX Measures and Calculated Columns
* Interactive Slicers and Filters
* KPI Cards and Performance Metrics
* Donut Charts, Bar Charts, Area Charts, and Tables
* Drill-Down and Cross-Filtering Functionality
* Business-Oriented Dashboard Design
